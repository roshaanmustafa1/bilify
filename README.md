Project Description
Bilify is a modern, AI-powered invoice generator built with Vue 3, TypeScript, and Tailwind CSS. It enables users to create professional invoices with a creative interface powered by generative AI capabilities.

Key Features
Vue 3 & TypeScript: Built with modern Vue 3 composition API and full TypeScript support
AI Integration: Powered by Google Generative AI for intelligent invoice generation and customization
Professional UI: Uses shadcn-vue components with Tailwind CSS for a polished, responsive interface
PDF Export: Generate and download invoices as PDF files (html2pdf.js)
Advanced Data Table: Built-in data table support via TanStack Vue Table for managing invoice line items
Icon Support: Multiple icon libraries (Lucide Vue, Iconify) for rich UI elements
State Management: Pinia for centralized state management
Customization: Component-driven architecture with Reka UI for flexible template design
Tech Stack
Frontend Framework: Vue 3, Vue Router
Build Tool: Vite 6
Styling: Tailwind CSS 4
UI Components: shadcn-vue, Reka UI
AI/ML: Google Generative AI API
Export: html2canvas-pro, html2pdf.js
Data Management: Axios, Pinia
Project Structure
The project is organized with:

Source files in the src/ directory
Component configuration via components.json (shadcn-vue setup)
Build configuration with vite.config.ts
TypeScript configuration for strict type checking
This is a perfect solution for businesses and freelancers who want to generate creative, customizable invoices with AI assistance, offering both a user-friendly interface and professional PDF output capabilities.
